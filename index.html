Functions

Functions are a named piece of code that can be used in other parts of your program. They are usually meant to perform one task well and help make code easier to read by substituting otherwise confusing code with a well named function call. 


function doTheThing() {
    console.log("It's working!");
}

Naming

Functions follow the same naming rules as variables. So they can contain letters (A-Za-z), numbers(0-9), underscores (_), and dollar signs ($). But a function name cannot start with a number.
It's convention that function names are written in camelCase, but this is not an enforced rule.

It is important that your function names are descriptive but not verbose, so a programmer can get a sense of what the function does without needing to pick apart the function itself.

Parameters

Parameters (or arguments) are variables passed into a function, that a function can use or operate upon.
You could make a function that accepts up to 255 parameters, though using a lot of parameters is sometimes a sign that your function may need to be split into a few smaller functions. 


function addNumbers(number1, number2) {
  let sum = number1 + number2;
  console.log(sum);
}

Functions also have access to an arguments variable that acts as an array holding all arguments passed into the function.
This is useful if you want to create a function that accepts an unknown number of parameters. 


function addNumbers() {
  let sum = 0;
  for(number of arguments) {
    sum += number;
  }
  console.log(sum);
}

Calling a function

To execute the code written in a function, you have to "call" (or invoke) the function. This is done by simply writing the function name followed by (). 


doTheThing();

You can pass arguments to a function by writing them as a comma separated list in between the (). 


addNumbers(5, 7, 3);

Return Values

Often times, you'll want to use a value from the function outside of the function. This can be achieved with a return statement. 


function addNumbers() {
  let sum = 0;
  for(number of arguments) {
    sum += number;
  }
  return sum;
}
let number = addNumbers(5, 7, 3);
/* Thanks to the return statement in our addNumbers function,
the result is now stored in a variable called number */

It's important to note that the function will stop execution as soon as the return statement is reached.
This can be used to exit a function early if needed. 


  function doTheThing(shouldStop) {
      if(shouldStop) {
        return;
      }
      console.log("It's working!");
  }

Objects

Objects are a container for a set of related, named variables called properties.
Objects are created (or instantiated) with a set of {} that wrap around all of its properties. 


let myCar = {
  make: 'Kia',
  model: 'Spectra',
  color: 'Blue',
  miles: 140000,
  isCool: false
}

Key/Value Pairs

Object properties are made up of what are called "key/value pairs". If you think of the object property as a variable, the key is the name of the variable and the value is the data that the variable holds.
The key is separated from the value using a :. 


let obj = {
  key: 'value'
}

When an object has more than one property (and they generally will), they are separated by a ,. 


//objects can be declared on one line
let obj2 = { key1: 1, key2: 2 }

//or on several
let obj3 = {
  key1: 1,
  key2: 2
}

Accessing a Property

We use what is called dot notation in order to use a property in an object. It is simply the name of the object followed by a . and then the name of the property you are trying to use. 


console.log( myCar.make ); // -> outputs 'Kia'
// now you can use object properties just like any other variable

We can also use what is called bracket notation in order to access a property. This is very similar to accessing an index in an array, but instead of putting a number between the [], we put the key of the property we are trying to access.
This can be useful if you have a string that holds a specific key for an object. 


let key = 'model';
console.log( myCar[key] ); // -> outputs 'Spectra'
console.log( myCar['model'] ); // -> ALSO outputs 'Spectra'

Common Methods

A method is simply a function that belongs to (is a property of) an object. You can access a method just like you would any other property of an object, and call it the same way you would any other function.

console.log() is an example of a method that belongs to the console object.

Object.keys(obj) Returns an array of keys of an object
Object.values(obj) Returns an array of values of an object
Object.assign(target, source) Copies the properties of the source(s) onto the target

Arrays

Arrays are a collection of data that can be thought of similar to a list. Arrays store your data (of any type) in the order you added it to the array, and work off of a 0 Based Index meaning that the first item is stored at index (position) 0.


let emptyArray = []; // this creates an empty array; an array with no data
let array = ['Hello', 'World', 2018]; // this creates an array that holds two strings and one number

Accessing an Array Item

You can access data stored in an array with bracket notation. Simply put the name of the array followed by [] with the index of the item you are trying to access in between the brackets.


console.log( array[0] ); // -> outputs Hello
